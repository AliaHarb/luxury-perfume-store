# luxury-perfume-store
A luxury, modern e-commerce web application specialized in premium product display and seamless shopping experiences. Built with a focus on high-end aesthetics, responsive layouts, and smooth user interactions.

---

##  Key Features

*   **UI/UX:** A highly customized, responsive interface featuring dynamic layouts tailored for high-end boutique stores.
*   **Dynamic Product Grid:** Optimized grid view with precise CSS image scaling to display dynamic external product assets flawlessly without layout breaking or unwanted padding.
*   **Advanced Brand Filtering:** Dynamic filtering architecture allowing users to browse specific brands smoothly with a robust fallback system for new database additions.
*   **Dynamic Pricing & Variants:** Real-time size selection updates that calculate variant pricing on the fly before cart updates.
*   **Cart Management:** Fully operational shopping cart system handling real-time additions, quantity modifications, and subtotal calculations.

---

## Tech Stack

*   **Frontend Framework:** React.js
*   **Styling & Design:** Tailwind CSS
*   **Fonts & Typography:** Premium Google Fonts integration
*   **Backend & Hosting:** Firebase Infrastructure
